Nom Time

ABOUT THE SITE

My third coding project and first experience working in a team. We built Nom Time in one week using Ruby on Rails and SQL. The idea behind the site is to create your own list of restaurants that you want to go to or that you’d recommend, and share these with others. You’d be able to see people's lists, bookmark them for safe keeping, and create a new list using spots from theirs. An interactive, easy way to stay up-to-date with the foodie scene!

Nom Time

ABOUT THE PROJECT

My third coding project, and first time working with a team! We built this in one week using Ruby on Rails and SQL. Nom Time is a site create your own list of restaurants that you want to go to or that you’d recommend, and share these with others. You’d be able to see other people’s lists and add places from theirs onto yours. Kind of like Spotify but with restaurants.
